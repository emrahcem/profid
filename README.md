ProFID (Protocol for Frequent Items Discovery)
======
This is a java source code for the implementation of a gossip based protocol to find frequent items in a network. As the gossip-based algorithm is not centralized, it does not have a single point of failure issue. ###How to Run
==============
You will simply run the profid.jar file. You can either double-click the profid.jar file or run the following command in the console

``java -jar /path/to/file/profid.jar``

to run the program.
